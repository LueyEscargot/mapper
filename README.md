### 2.2. 数据结构说明

#### 2.2.1. 会话结构体

会话结构体 (SessionBlk_t) 用于保存与一个会话相关的所有信息，其中包括：

- 状态机
- 套接字：包括南向（至客户端）与北向（至后台服务器）
- 缓冲区：包括南向缓冲区及北向缓冲区
- 其他相关数据对象（如：同步对象等）

具体数据结构定义请参阅相关代码，在此不再复述。
